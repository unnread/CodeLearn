# CodeLearn

CodeLearn est une plateforme d'apprentissage du développement web conçue pour les débutants.

Le projet est accessible :

-  Depuis un navigateur web
-  Comme application de bureau grâce à Electron.js

## Fonctionnalités

- Apprentissage du HTML
- Apprentissage du CSS
- Apprentissage du JavaScript
- Interface simple et moderne
- Navigation rapide
- Léger et facile à utiliser

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- Electron.js

## Installation

### Cloner le projet

```bash
git clone https://github.com/unnread/CodeLearn.git
```

### Lancer la version Web

Ouvrez simplement le fichier `index.html` dans votre navigateur.

### Lancer la version Electron

```bash
cd CodeLearn
npm install
npm start
```

## Objectif

CodeLearn a pour objectif de rendre l'apprentissage du développement web plus simple, plus accessible et plus pratique grâce à des explications claires et des exemples concrets.
